Get started in 60 seconds

Three commands. That's it.

1

Install

$ npm install -g relay-protocol
2

Start

$ relay start

Opens dashboard at localhost:8787

3

Register your agent

import { quickConnect } from 'relay-protocol'

quickConnect({
  name: 'MyAgent',
  capabilities: ['web_search'],
  handler: async (task, params) => {
    return { success: true }
  }
})
